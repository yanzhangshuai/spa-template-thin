vue3 + Vite + Typescript template

### 命令

```
pnpm dev                     开发环境

pnpm build                   生产环境
pnpm build:br                生产环境, 文件压缩.br格式
pnpm build:gz                生产环境, 文件压缩.gz格式

pnpm deploy                  构建 Docker 部署镜像

pnpm test                    单元测试
pnpm lint                    eslint、stylelint

pnpm report                  分析生产环境编译结果
pnpm report:test             分析单元测试结果
pnpm report:windi            分析 windicss 使用结果

```
